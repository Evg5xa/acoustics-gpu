# GPU-ускоренное моделирование акустики помещений: Ray Tracing + Image Source

Библиотека для высокопроизводительного моделирования распространения звука в помещениях с использованием GPU-ускоренной трассировки лучей (Ray Tracing) и метода зеркальных источников (Image Source). Поддерживает учёт частотных характеристик материалов и визуализацию энергетических карт.


## Основные возможности

- **GPU-ускорение на базе PyTorch** — ускорение в 10–100+ раз по сравнению с CPU (масштабируется с объёмом данных).
- **10 частотных полос** — моделирование в октавных полосах от 31.5 Гц до 16 кГц.
- **8+ реальных материалов** — коэффициенты поглощения из таблицы компании Фронт Акустик (источник: https://frontacoustic.ru/assets/foruser/blog/Tablezvuk/Таблица%20коэффициентов%20звукопоглощения.pdf).
- **Два метода моделирования**:
  - Трассировка лучей (Ray Tracing) — для поздней реверберации и диффузного поля.
  - Метод зеркальных источников (Image Source) — для точных ранних отражений.
- **Визуализация** — энергетические карты для всех частотных полос и частотные характеристики в точках микрофонов.
- **Гибкость** — автоматическое переключение между CPU и GPU, интерактивный ввод параметров.

## Интерактивный ввод параметров

Размеры комнаты

Материалы стен

Количество лучей и отражений

Расположение источника и микрофонов

## Библиотека материалов

Бетон, кирпич, штукатурка

Ковёр, линолеум, ламинат

Гипсокартон, дерево, стекло

Акустический пенопласт, бархат и другие

## Результаты

Энергетические карты для всех 10 частот

Частотные характеристики в точках микрофонов

Статистика распределения энергии

## Требования
Python 3.8+

Видеокарта с поддержкой CUDA (опционально, работает и на CPU)

PyTorch с поддержкой CUDA (Для GPU)
